Steps to configure Maven POM file
1. Install JDK and set JAVA_HOME
2. Install Maven and set home variable. MAVEN_HOME
3. Install tomcat and setup users.
4. Write POM.
* Download dependencies 
* Compile using Compiler plugin
* Run unit test cases using surefire plugin
* Generate code coverage report using Cobertura plugin
* Package war
* Deploy war to Tomcat using Tomcat plugin 
5. While configuring POM, configure all the plugins.
6. In Global Maven settings file, configure Tomcat Server.
